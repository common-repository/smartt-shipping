This plugin allows you to add the SMARTT Shipping features in your website.

== Description ==

= BENEFITS = 
* Get discounted shipping rates. 

* Send tracking info to your customers. 

* Automatically import WordPress orders.

* Pre-fill customs documentation for international orders.

* Compare real-time Courier and LTL Carrier options.

* Option to display rates on checkout for your customer’s preferences. (We import the customer’s selected shipping choice, which you can then map to the appropriate shipping carrier or service).

* Create shipping labels. 

* Automatically fulfill orders & update the carriers' tracking URL for WordPress orders. 

= HOW IT WORKS =
* After installing the SMARTT Shipping plugin, you will be prompted for an API Key which you can request from your Account Manager or techsupport@completeshipping.ca. 

* Add the API key to your WooCommerce settings > Shipping > SMARTT Shipping page.

* Immediately access discounted carrier rates and print shipping labels.

* To use SMARTT Shipping on your eCommerce website.

== Installation ==

1. Upload the plugin files to /wp-content/plugins OR you can search “SMARTT Shipping” on your WordPress Dashboard.(WordPress Dashboard > Plugins > Add New) 
	
2. Once you have found the app, select ‘Install‘ and then ‘Activate‘ 

3. The SMARTT Shipping app will now be added to your WooCommerce Shipping Settings on the WordPress Dashboard OR You can also redirect to Shipping Settings page by click on settings menu under the plugin page. 

== Screenshots ==

1. On WordPress, go to WooCommerce > Settings > Shipping > SMARTT SHIPPING.

2. Insert all the default values like (Weight, height, length, package type, product type, etc.).

3. Go to the Products page to adjust package details and values.

4. When you are ready to ship an order, go to the Order page and select “Ship Order”.

5. Adjust shipping information & Generate shipment.

6. The system will then calculate the shipping costs from the carrier(s).

7. Once your shipment is generated, you will receive all the details about the order's shipment.

8. You can manage multiple warehouse addresses by going to “Smartt Shipping Warehouse Address”.

9. Choose the warehouse address you want to ship the order from.

10.	To customize the carriers so that only your preferred ones appear during checkout, select your preferred carriers on the “Smartt Shipping Preferred Carriers” tab.

11. Only the selected carriers will be visible on the Cart page. If none are selected, all carriers will appear.

12. Only the selected carriers will be visible on the Checkout page. If none are selected, all carriers will appear.

== Frequently Asked Questions ==

**Q: Where can I find the API key for the plugin activation?**

A: Your Account Manager at Complete Shipping Solutions is able to provide you with an API key. 

**Q: Do I need to know any coding to use this plugin?**

A: No, we have created an easy-to-use plugin that requires zero coding knowledge. 

**Q: Does SMARTT Shipping work with any WordPress theme?**

A: Yes, the SMARTT Shipping plugin works with any theme embedded within the WooCommerce plugin.
